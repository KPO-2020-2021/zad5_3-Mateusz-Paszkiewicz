Detailed Description

template<typename T, unsigned int dime>
class Matrix< T, dime >

Klasa macierzy kwadratowej dowolnego rozmiaru, jej metody oraz funkcje.

Klasa zawiera definicje, metody i funkcje macierzy potrzebne do zrealizowania obrotu 3D za pomoca lacza do GNUplota.
