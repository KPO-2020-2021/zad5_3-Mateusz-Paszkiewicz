Detailed Description

Klasa SceneObject definiuje ogolnie pojety objekt na scenie za pomoca zestawu danych w postaci Cuboid.

Klasa ta uzywa klasy cuboid z zapetlonymi punktami aby odwzorowac konkretna przeszkode na scenie.
